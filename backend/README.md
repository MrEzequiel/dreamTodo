## 📜 Colaboração

```bash

 #Clone o repositório
 $ git clone https://github.com/MrEzequiel/simple-to-do.git

 #Entre na pasta do projeto
 $ cd simple-to-do
 $ cd backend

 #Instale as dependências
 $ yarn install

 #Inicie a aplicação
 $ yarn dev

```
